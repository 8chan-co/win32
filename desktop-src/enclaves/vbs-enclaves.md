# Virtualization-based security (VBS) enclaves

Virtualization-based security (VBS) enclaves are used to create trusted execution environments. VBS enclave is a software-based solution with no special hardware dependencies, see [Virtualization-based Security (VBS)](/windows-hardware/design/device-experiences/oem-vbs) for more information.
